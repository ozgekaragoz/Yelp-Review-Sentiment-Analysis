# Yelp Review Sentiment Analysis

## Project Description
This project was completed as part of a homework assignment and focuses on **sentiment analysis of Yelp reviews**.

The main objective is to analyze customer reviews and classify them based on their sentiment using **Natural Language Processing (NLP)** techniques. The study explores how textual review data can be transformed into meaningful numerical representations and used for sentiment classification.

---

## Dataset
The dataset consists of Yelp customer reviews containing textual feedback and corresponding sentiment labels.

> ⚠️ The dataset is **not included** in this repository.  
> The notebook is structured so that the analysis can be reproduced by providing the dataset in the expected format.

---

## Methodology
The following steps were applied in the analysis:

- Text preprocessing  
  - Lowercasing  
  - Removal of punctuation and special characters  
  - Tokenization  
  - Stopword removal  

- Feature extraction  
  - Text vectorization using common NLP techniques  

- Sentiment analysis  
  - Classification of reviews based on sentiment polarity  

- Evaluation  
  - Model performance analysis using appropriate evaluation metrics  

All steps are implemented and explained within the Jupyter Notebook.
